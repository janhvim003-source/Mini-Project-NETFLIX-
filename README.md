# Netflix Data Analysis & Business Insights

This project involves a comprehensive Exploratory Data Analysis (EDA) of Netflix's content library. The goal is to uncover trends in movies and TV shows, analyze global content distribution, and provide data-driven insights to guide business growth.

## 🚀 Key Objectives
* **Content Distribution:** Analyzing the ratio between Movies and TV Shows.
* **Top Contributing Countries:** Identifying which countries produce the most content for Netflix.
* **Genre Analysis:** Understanding the most popular genres and categories.
* **Rating Insights:** Evaluating the target audience (TV-MA, TV-14, etc.) across different regions.

## 🛠️ Tech Stack
* **Language:** Python
* **Libraries:** Pandas, NumPy, Matplotlib, Seaborn
* **Platform:** Google Colab

## 📈 Key Insights
* Identified the top 10 countries with the highest content volume.
* Visualized the growth of Netflix content over the last decade.
* Highlighted the most common ratings to determine the platform's primary audience.

## 📋 How to Run
1. Open `NETFLIX (3).ipynb` in Google Colab.
2. Upload the `netflix_titles.csv` dataset when prompted (or ensure the path is correct).
3. Run all cells to generate the visualizations and statistical summaries.
